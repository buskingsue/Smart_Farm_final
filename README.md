## 라즈베리 파이와 STM32 F411RE 보드를 이용한 스마트 팜 프로젝트
### 1. 기능
- 온도센서 습도센서 조도센서 값 입력
- 팬 물펌프 LED 동작
- 라즈베리파이 파이차트 출력
### 
### 2. 사용한 부품
- 온도 습도 센서, 조도 센서
- LCD , 물펌프
- `HC-06` 블루투스 모듈
